## Building / running

- Have packages installed (`npm i / yarn`)
- `npm run build` - builds project in `./dist`
- `npm start` - starts `./dist/main.js`, requires `npm run build` to have been
  run previously.
- `npm run dev` - starts real-time build for testing

### Passing arguments

- with npm: use `npm run _\<script\>_ -- _\<args\>_
- with yarn: `yarn _\<script\>_ _\<args\>_
